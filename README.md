## New: Camera Control Support (iPhone 16+)

Now with **experimental Camera Control integration** for scrolling through the carousel using the physical Camera Control button!

**Features:**
- Light press Camera Control → activate carousel slider overlay
- Slide finger on Camera Control → scroll through posters
- Full synchronization with on-screen swipes and iPod wheel
- Automatic UI hiding in fullscreen mode

**Requirements:**
- iPhone 16, 16 Plus, 16 Pro, or 16 Pro Max
- iOS 18.0+
- Camera permission (for AVCaptureSession API)

See [CAMERA_CONTROL_GUIDE.md](CAMERA_CONTROL_GUIDE.md) for detailed setup and usage instructions.
